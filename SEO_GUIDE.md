# SEO Guide for Lara Advertising & Printing

This project is configured for SEO around the main service themes: مطابع الرياض, خدمات الطباعة في الرياض, طباعة بنرات الرياض, طباعة ستكرات الرياض, banner printing Riyadh, and sticker printing Riyadh.

## 1) Google Search Console Setup
1. Open Google Search Console.
2. Add the property for:
   https://lara-advertising-printing.vercel.app/
3. Use the URL-prefix property if you want the fastest setup.
4. Verify ownership using the recommended method.

## 2) Submit the Sitemap
1. In Search Console, open Sitemaps.
2. Submit:
   https://lara-advertising-printing.vercel.app/sitemap.xml
3. Wait for Google to crawl it.

## 3) Request Indexing
1. Use the URL Inspection tool.
2. Paste:
   https://lara-advertising-printing.vercel.app/
3. Click Request Indexing after any major content update.

## 4) What Google Should Read
The site is now configured with:
- Strong Arabic title focused on service intent.
- Meta description for printing and advertising in Riyadh.
- Canonical URL pointing to the real production domain.
- Open Graph and Twitter metadata for social sharing.
- JSON-LD structured data using PrintingService.
- Robots and sitemap files for indexing.

## 6) How to Improve Rankings Over Time
- Add real photos and service case studies.
- Create FAQ content answering customer questions.
- Add more localized phrases for Riyadh neighborhoods.
- Build backlinks from social profiles, directories, and business listings.
- Keep titles and descriptions unique if you add more pages later.
- Keep performance high: compressed images, lazy loading, and simple layouts.

## 7) Important Notes
- Do not use "noindex" on production pages.
- Keep the canonical URL on the production domain.
- Update the sitemap when you add new pages.
- Re-submit the sitemap if the website structure changes.
